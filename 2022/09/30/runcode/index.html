PHP中一些任意代码执行的相关函数介绍

注意:在以下函数中,回调函数一般总是会被执行

assert :

如果 assertion 是字符串，它将会被 assert() 当做 PHP 代码来执行。

PHP 5 和 7

assert(mixed $assertion, string $description = ?): bool

PHP 7

assert(mixed $assertion, Throwable $exception = ?): bool

示例:

assert(‘phpinfo()’);
